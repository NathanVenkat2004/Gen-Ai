# 📘 NotebookLM - AI Powered Document Assistant

## 🚀 Features

* 📄 Upload PDFs and ask questions
* 🤖 RAG (Retrieval Augmented Generation)
* 🌐 Web Search using Tavily API
* 📝 Save answers as Notes
* 🔍 Source-based responses

## 🧠 Tech Stack

* Streamlit
* LangChain
* ChromaDB
* Ollama (Embeddings)
* Tavily API

## ⚙️ Setup Instructions

```bash
git clone https://github.com/NathanVenkat2004/Gen-Ai
cd Gen-Ai
pip install -r requirements.txt
streamlit run app.py
```

## 🔑 API Key Setup

Add your Tavily API key in chat.py:

```python
TavilyClient(api_key="your_api_key_here")
```
